# Voice-cloning-system-using-Fastspeech2-ECAPA-TDNN-English-_

  1. Non-autoregressive model
  2. Faster training and inference
  3. Predicts mel-spectrograms directly

🔹 Speaker Embedding Model: ECAPA-TDNN
 1. Extracts speaker-specific features
 2. Helps clone voice identity

🔹Dataset
 1. LibriTTS Dataset
 2. High-quality speech dataset derived from audiobooks
 3. Multiple speakers with clean recordings
 4. Suitable for TTS and voice cloning tasks
